## 安装

* npm 安装

  ```bash
  npm install pks-cli
  ```

* yarn 安装

  ```bash
  yarn add pks-cli
  ```

### yarn 安装

## 构建项目

```bash
# 构建测试版
npm run dev

# 构建正式版
npm run build

```
